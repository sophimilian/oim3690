## Class Notes 
-  what is an API?
    - application programming interface
    - a way for programs to talk to each other 
    ex: restaurant
        - menu--API documentation 
        - waiter--API
        - kitchen--server
    - common API's: weather, GitHub, movie, currency (our browser already uses API constantly like loading tweets, and checking email) 
- sending data uses fetch. 
- protecting API:
    - the main problem is: some APi's require APi key to indentify you (if we push it to GitHub, anyone can steal my API)
- Even with .gitignore, anyone who opens your deployed site can see the key in DevTools (Network tab or Sources).
- for real apps: use a backend server that keeps the key secret and proxies requests. 

JAVASCRIPT: IMAGE PROCESSING 
- images are made of pixels
- every digital image is a grid of tiny colored squares called pixels 
- A 1920x1080 image has over 2 million pixels
- Each pixel has a color value
How they store : 
Value	Range	Meaning
R	0-255	Red intensity
G	0-255	Green intensity
B	0-255	Blue intensity
A	0-255	Alpha (transparency)
