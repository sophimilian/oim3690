- selector specificity (from lowest to highest priority)
    - element sectors: p, div, h1
    - class selectors: .intro, .btn
    - ID selectors: #header, #main
    - inline styles: style="..."
- rememeber the 3 ways to add CSS to a page:
    1. internal: in the same file where you use the style tag to start
    2. inline: add the style inside the tab 
    3. in another document 
(always better to add everything in one place instead of within each line)
- cascading: where multiple rules exist for each element
- elements in a box:
    - content: the actual content
    - padding: space inside the border
    - border: the edge
    - margin: space outside the border 
- basic selectors:
    - element: targets all elements of a type --> p{}
    - class: targets all elements in a class --> .intro{}
    - ID: targets elements with ID --> #header{}
    - universal: targets ALL elements --> *{}
